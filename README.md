# pillme 💊

### Why
1. **놓치지 않도록** 
- 사용자가 약을 규칙적으로 먹을 수 있게 푸시알람을 보낸다.
3. **이미 먹었나? 잊지 않도록**
- 사용자가 먹은 / 잊은 약을 알 수 있도록 한다.
5. **기록으로 남겨두자**
- 사용자의 복용순응도를 달력으로 확인하도록 한다.

### 필요 기능
- [ ] Main화면 - 오늘 하루 먹어야하는 약 / 먹은 약 표시
- [ ] 복용약 추가 화면 - step by step으로 약을 쉽게 추가할 수 있도록
- [ ] 달력 화면 
- [ ] 시간대에 맞게 푸시 알람

### 추가 구현
- [ ] 애플 워치로 알람 보내기
- [ ] 나만의 custom 시간을 설정할 수 있도록 하기
